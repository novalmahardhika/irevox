## How to run this code ?

### Clone repo

```bash
git clone  https://github.com/novalmahardhika/irevox.git
```

### Go directory

```bash
cd irevox
```

### Go to file client and server

```bash
cd server && cd client
```

### Install package in both of files

```bash
npm i
```

### Migrate db in server

```bash
npm run db:migrate
```

### Run in a server and client

```bash
npm run dev
```

## Command

### Generate Seeders

```bash
npm  run db:seed
```

### Reset db

```bash
npm  run db:reset
```
